Full-stack Web Developer

Az Adecco pénzügyi szektorban működő partnere részére, budapesti munkavégzésre PHP és JavaScript ismerettel full-stack web developer munkatársat keres.

Feladatok:

  • a vállalati web alapú keretrendszer továbbfejlesztése,
  • különböző webes modulok/felületek fejlesztése a belső osztályok részére,
  • új web alapú rendszerek tervezése, kivitelezése, a csapat tagjaként projektfeladatok széleskörű menedzselése.

Elvárások:

  • minimum 3 év webfejlesztésben szerzett tapasztalat,
  • készségszintű JS/AJAX ismeretek,
  • erős szintű (OOP) PHP 5 hozzáértés, valamely PHP-keretrendszerben való jártasság (pl. Yii, Symfony, Zend),
  • event-driven, MVC szemlélet,
  • webes biztonsági alapismeretek,
  • gyors hibakereső és hatékony problémamegoldó képesség.

Előnyt jelent:

  • SOAP, REST ismeret,
  • adatbáziskezelési, -programozási készség,
  • Node.js tapasztalat,
  • verziókezelő-rendszerek ismerete (SVN, Git),
  • debugeszközökben való jártasság (Firebug, Xdebug).

Amit kínálunk:

  • lehetőség új, innovatív projektekre önálló, kreatív kidolgozással,
  • rendszeres belső és külső képzések,
  • egyénre szabott karrier út, oktatási terv, félévenkénti karriermenedzsment.
