PHP-webfejlesztő, Budapest

A Violet Consulting piacvezető, nemzetközi háttérrel rendelkező partneréhez hazai és külföldi projektekre tapasztalt PHP-webfejlesztő munkatársat keres Postgres és PL/SQL ismerettel.

Feladatok:

  • HTML és XML/JSON API backend-fejlesztés,
  • tapasztalattól függően a tervezésben való részvétel.

Elvárások:

  • minimum 2-3 év PHP webfejlesztői tapasztalat,
  • PHP 5 magas szintű ismerete, objektumorientált tapasztalat,
  • Postgres adatbázis és PL/SQL nyelv ismerete,
  • HTML, CSS, SASS, JavaScript, jQuery ismerete,
  • Unix parancsok, Git, Vim használata,
  • kommunikációképes angol nyelvtudás,
  • precíz, rugalmas, csapatjátékos személyiség.

Előnyt jelent:

  • felsőfokú szakirányú végzettség,
  • PHPUnit ismerete, automatikus tesztek készítésében szerzett tapasztalat,
  • Bootstrap, CoffeeScript tapasztalat,
  • Ruby, C nyelvek ismerete.

Amit kínálunk:

  • azonnali kezdés,
  • kiváló megközelíthetőségű iroda,
  • modern irodai környezet, korszerű MacBook Pro és Toshiba notebookok,
  • korszerű fejlesztési módszertanok: scrum, agilis eszközök,
  • inspiráló szakmai légkör,
  • fiatalos, nyitott vállalati kultúra,
  • versenyképes jövedelem és juttatási csomag,
  • cafeteria szolgáltatások,
  • végtelen mennyiségű kávé és tea.
